# Windows-IT-Deployment-Inventory-Automation-PowerShell-
# IT Automation Toolkit (PowerShell)

This project includes PowerShell scripts designed to automate common IT support tasks.

## Features

### 1. Inventory Script
- Collects system information:
  - Hostname
  - Serial Number
  - OS
  - IP Address
  - MAC Address
- Saves output to a structured file

### 2. Application Deployment Script
- Automatically installs common applications using winget
- Runs with administrative privileges
- Supports silent installation

## Use Case
Designed for IT Support and Help Desk environments to automate device setup and asset tracking.
